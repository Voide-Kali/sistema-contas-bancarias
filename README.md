# Sistema de Contas Bancárias

Implementação em JavaScript de contas bancárias com herança simples, usada para estudo de orientação a objetos.

## O que tem

- `Conta.js`: classe base
- `ContaCorrente.js`: conta corrente
- `ContaPoupanca.js`: conta poupança
- `ContaSalario.js`: conta salário
- `index.js`: ponto de entrada / testes de uso

## Estrutura

```text
Bancárias/
├── Conta.js
├── ContaCorrente.js
├── ContaPoupanca.js
├── ContaSalario.js
├── index.js
└── src/
```

## Como usar

Este repositório é pequeno e não exige instalação formal. Abra `index.js` ou importe as classes nos seus testes.

## Objetivo

O foco é praticar:

- encapsulamento;
- herança;
- polimorfismo;
- reutilização de código.
